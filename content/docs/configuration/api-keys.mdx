---
title: Setup API Keys
description: Setting up API keys to send emails.
---

Go to the [API Keys](/api-keys) page and create your API key by following these steps:

## Steps:

<Steps>
### Sign up/ Login to your Freesend account

### Navigate to the API Keys section in the Navbar

<img
  className="w-[200px]"
  src="/images/api-keys-navigation.png"
  alt="API Key Navigation"
/>

### Create a new API Key by clicking on "Add Mail Server"

### Fill in the details for your key

<img
  className="w-[300px]"
  src="/images/add-api-key-details.png"
  alt="Add API key details"
/>
<Callout type="warning">
  If you have not configured any mail servers yet, refer to [Setup Mail
  Servers](/docs/configuration/mail-server).
</Callout>
### Done!

You can now [send emails](/docs/api/send-email) from the configured SMTP Server!

</Steps>
